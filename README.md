# ayx-portal-archive

## 项目简介

`ayx-portal-archive` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为静态页面的集中存储与展示空间。

## 目录说明

```
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面所需的资源文件（如图片、样式、脚本）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- **pages/**：每个独立 HTML 页面可放置于此目录下，建议按功能或时间命名。  
- **assets/**：页面引用的外部资源统一存放，便于管理。  
- **index.html**：可自定义仓库入口页面，用于导航或说明。

## 页面归档说明

归档页面应遵循以下基本规范：

- 每个页面应保持独立，避免依赖外部不可控资源（建议使用相对路径或内联资源）。  
- 页面内容应合法合规，不包含恶意代码、侵犯隐私或违反平台政策的信息。  
- 建议在页面头部添加简要注释说明用途或归档时间。
